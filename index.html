GOLANG CALCULADORA

Calculadora de consola que hace las operaciones basicas

Versión en Español | English Version

Enlace al repositorio:

Funcionalidades 📋

La calculadora permite realizar operaciones básicas entre dos números desde la consola:

  • ➕ Suma
  • ➖ Resta
  • ✖️ Multiplicación
  • ➗ División

Descripción 📌

La calculadora está diseñada para ejecutarse en la terminal y recibir la entrada del usuario, procesando las operaciones mediante funciones separadas para mantener un código limpio y modular.

Comenzando 🚀

  1. Clona el repositorio en tu máquina local:
    2. git clone https://github.com/12345star/calculadora-golang.git
  2. Navega al directorio del proyecto:
    3. cd calculadora-golang

Dependencias 📦

Documentacion Dependencias.

  • Golang
  • UNIT (Testify)
  • Modulo tidy

Asegúrate de tener instalado Go (v1.18 o superior) en tu sistema.

go mod init calculadora
go mod tidy
go get github.com/stretchr/testify/assert

Uso ⚙️

  1. Ejecuta operaciones desde consola:
    • Suma: 
      go run index.go 5 + 3
    • Resta: 
      go run index.go 5 - 3
    • Multiplicación: 
      go run index.go 5 * 3
    • División: 
      go run index.go 6 / 3
    • Estructura:
      go run index.go <num1> <operador> <num2>

🧪 Pruebas

Para ejecutar las pruebas unitarias:

go test ./...
