🔢 Calculadora en Go

Calculadora de operaciones básicas hecha en Golang (Go)

🧮 Funcionalidades

La calculadora permite realizar operaciones básicas entre dos números desde la consola:

  • ➕ Suma
  • ➖ Resta
  • ✖️ Multiplicación
  • ➗ División

📋 Descripción

La calculadora está diseñada para ejecutarse en la terminal y recibir la entrada del usuario, procesando las operaciones mediante funciones separadas para mantener un código limpio y modular.

🛠️ Requisitos de instalación

Asegúrate de tener instalado Go (v1.18 o superior) en tu sistema.

go mod init calculadora
go mod tidy
go get github.com/stretchr/testify/assert

⚙️ Instalación y uso

  1. Clona el repositorio:
    2. git clone https://github.com/12345star/calculadora-golang.git
  2. Ingresa al directorio:
    3. cd calculadora-golang
  3. Ejecuta operaciones desde consola:
    • Suma: go run index.go 5 + 3
    • Resta: go run index.go 5 - 3
    • Multiplicación: go run index.go 5 * 3
    • División: go run index.go 6 / 3

🧪 Pruebas

Para ejecutar las pruebas unitarias:

go test ./...

📂 Estructura del proyecto

.
├── index.go                # Punto de entrada principal
├── go.mod                  # Módulo de Go
├── go.sum                  # Dependencias
├── model/
│   └── operaciones.go      # Lógica de operaciones matemáticas
├── testing/
│   └── operaciones_test.go # Pruebas unitarias
└── view/
    └── view.go             # Interfaz de entrada/salida en consola
